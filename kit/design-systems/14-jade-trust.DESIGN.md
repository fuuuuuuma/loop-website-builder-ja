# DESIGN.md — ジェイド・トラスト（Jade Trust）

> **これは何**：Webサイトの「配色・フォント・型」をひとまとめにした正典です。このキットの `site-builder` に
> 「この DESIGN.md に従って作って」と渡すだけで、この見た目で統一されたサイトになります（スライドにも流用可）。
> 番号 14 / 30 ／ カテゴリ: お金・信頼 ／ モード: ライト（明背景）

## 使いどころ
- 汎用。トンマナに合わせて調整。
- 迷ったら、参考サイト（`../reference/reference-sites.md`）で近いトンマナを1〜2本選び、当て先にする。

## カラートークン（意味ベース）
| トークン | 役割 | 値 |
|---|---|---|
| `--brand` | 主色（見出し・強調） | `#0F766E` |
| `--brand-deep` | 主色の濃い版（小見出し・帯） | `#0B5953` |
| `--accent` | 差し色（ボタン地・枠・バッジ） | `#D4AF37` |
| `--accent-deep` | 差し色の濃い版（ホバー） | `#A8841E` |
| `--bg / --paper` | 背景 / カード白面 | `#F1FAF8 / #FFFFFF` |
| `--ink / --ink-mid` | 本文 / 補助テキスト | `#0F4A45 / #436B66` |
| `--ink-sub` | 極淡（**罫線など非テキスト専用**） | `#83A6A1` |
| `--on-accent` | accent 地の上の文字色 | `#0F4A45` |

## そのまま貼り付け（`:root`）
サイトの `:root{ ... }` をこのブロックで丸ごと置き換えてください。

```css
:root{
  --brand:#0F766E; --brand-deep:#0B5953; --brand-soft:rgba(15,118,110,0.08); --brand-line:rgba(15,118,110,0.22);
  --accent:#D4AF37; --accent-deep:#A8841E; --accent-soft:rgba(212,175,55,0.16);
  --bg:#F1FAF8; --paper:#FFFFFF; --ink:#0F4A45; --ink-mid:#436B66; --ink-sub:#83A6A1;
  --line:rgba(15,74,69,0.12); --on-accent:#0F4A45; --canvas:#D6ECE8;
  --head:'Noto Sans JP';
}
```

> 見出しフォント：**Noto Sans JP**。`<head>` に追加：
> `<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+JP:wght@400;500;700;900&display=swap" rel="stylesheet">`

## Webサイトのセクション型（このテーマの基本ブロック）
1. ヒーロー（大見出し＋一言サブ＋一次CTAボタン）
2. 信頼の証（ロゴ群／実績数字のバッジ列）
3. 課題提起（読み手の「困った」を1〜3行で）
4. 特徴3本柱（カード×3・アイコン＋見出し＋説明）
5. 使い方3ステップ（矢印フロー）
6. before / after または 比較表（自分 vs 従来）
7. 料金プラン（3カード・推しを1つだけ強調）
8. お客様の声 ／ FAQ（アコーディオン）
9. 最後のCTA（もう一度だけ背中を押す・ボタン1つ）
10. フッター（リンク・運営者情報・特商法/プライバシー）

## コンポーネントの当て方（トークン → UI）
- ボタン（一次）：地 `--accent`／文字 `--on-accent`。ホバーで `--accent-deep`。**accent地に白文字は禁止**。
- ボタン（二次）：枠 `--brand`／文字 `--brand-deep`／地は透明。
- 見出し `--ink`・`--brand` ／ 小見出し `--brand-deep` ／ 本文 `--ink-mid`。
- カード：地 `--paper`／枠 `--line`／影は薄く。強調カードだけ枠 `--brand`。
- リンク：`--brand-deep`（下線）。
- セクション背景は `--bg` と `--paper` を交互に。区切り線は `--line`。
- タップ領域は 44px 以上。フォーカスリングは `--accent`。

## コントラスト規約（必守）
- `--accent`／`--accent-deep` は **枠線・地色・バッジ・ボタン地 専用**。**小さい文字色に使わない**。
- 文字色は `--ink` / `--ink-mid` / `--brand-deep`（背景に対して WCAG AA を確保）。
- ボタン・バッジは **accent 地 ＋ `--on-accent` の文字**。accent 地に白文字は禁止。
- `--ink-sub` は罫線・区切りなど **非テキスト専用**。本文・キャプションには使わない。
- 表の見出しは **濃い地＋白文字**。

## 禁止表現（誇張・断定保証）
`必ず / 絶対 / 確実 / 裏技 / 神 / ヤバい / 衝撃 / 100%稼げる` は使わない。効果は中立に（「〜も目指せる」）。景表法・薬機法・特商法・ステマ規制に配慮。
