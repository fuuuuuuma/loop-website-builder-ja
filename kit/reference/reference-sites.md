# 参考Webサイト集 — トンマナの「当て先」を探す

作る前に、**近い雰囲気のサイトを1〜2本決める**。それが「デザインチェッカー」の採点基準になり、ループが速く収束する。
ギャラリーで探し、良いものはスクショして `site-spec.md` に URL を貼る。**レイアウトのコピーではなく、トンマナの参考**に。

> リンクは変わることがあります。開けないときはサイト名で検索してください。

## 使い方（3ステップ）
1. ギャラリーで、**作りたいサイトに近いもの**を1〜2本選ぶ（業種・雰囲気で絞る）
2. その URL を `site-spec.md` の「参考サイト」に貼る＝**トンマナの基準**に
3. `design-systems/` から近い配色の DESIGN.md を1枚選び、`:root` を固定して寄せる

> 注意: 参考はあくまで**トンマナ**。デザイン・文言・画像をそのまま流用しない（著作権）。自分の目的とコピーで作る。
